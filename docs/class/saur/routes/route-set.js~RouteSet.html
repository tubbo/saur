    RouteSet

    RouteSet defines the routing DSL used by the top-level application router. It uses Oak.Router under the hood, but pre-fills information based on the current context and whether you've selected a controller. Controllers are selected by providing a resources() route, and base paths can be selected by themselves by providing a namespace().
